# Thư mục chứa Video Giám khảo ảo (Speaking Examiner Video)

Khầy Được ơi, thư mục này được dùng để chứa file video của giám khảo ảo đọc câu hỏi phần thi Nói (Speaking).

### Hướng dẫn sử dụng:
1. Chuẩn bị file video đọc câu hỏi thi Nói định dạng `.mp4`.
2. Đổi tên file video thành: `speaking_examiner.mp4`.
3. Copy/Paste file video đó vào thư mục này (`public/video/speaking_examiner.mp4`).
4. Khi chạy ứng dụng, hệ thống sẽ tự động nhận diện và trình chiếu video này theo đúng timeline 22 câu hỏi (từ 0s đến 263s).

*Lưu ý: Nếu không có file video này trong thư mục, hệ thống sẽ tự động kích hoạt chế độ dự phòng đọc câu hỏi bằng giọng đọc nhân tạo (Speech Synthesis - TTS) để đảm bảo bài thi không bị gián đoạn.*
